# Contributing

Contributions are welcome from anyone - whether you're an expert in scrollytelling or simply interested in learning and experimenting. 

## How to Contribute

- For making improvements/fixes to the demo or adding new ideas and techniques to the README, feel free to open a pull request.
- Issues can be opened for suggestions or questions.

# Guidelines
- Please use comments in your code for better readability - remember that it's purpose is as a demo for the notes.
- Keep contributions simple and focused. Complex combinations of techniques are not necessary.
- If you're adding to the notes, keep things simple and organized.
